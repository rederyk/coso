## Step-by-Step Guide

### Method 1: LVGL Online Font Converter (Recommended)

#### 1. Open the Converter
Go to: https://lvgl.io/tools/fontconverter

#### 2. Download Noto Emoji Font
Download from: https://github.com/googlefonts/noto-emoji/raw/main/fonts/NotoEmoji-Regular.ttf
(Or use Noto Color Emoji for colored emojis)

#### 3. Configure for Each Size

**For 14px:**
- Name: `emoji_14`
- Size: `14`
- Bpp: `4` (4 bits per pixel - good quality, moderate size)
- TTF/OTF file: Upload `NotoEmoji-Regular.ttf`
- Range: Choose "Symbols" and paste the Unicode ranges below
- Font fallback: `lv_font_montserrat_14` (optional)

#### 5. Repeat for Other Sizes
Repeat steps 3-4 for:

- `emoji_48.c` (size 48)
- `emoji_96.c` (size 96)

### Method 2: Command Line (Advanced)

Install lv_font_conv:
```bash
npm install -g lv_font_conv
```

Run for each size:
```bash
lv_font_conv \
  --font NotoEmoji-Regular.ttf \
  --size 14 \
  --bpp 4 \
  --format lvgl \
  --range 0x20-0x7E,0x1F300-0x1F6FF \
  --output emoji_14.c \
  --force-fast-kern-format
```
